Computer Systems Fundamentals


Overview of the Unit

This unit provides an introduction to the fundamental principles and mathematics underpinning the design and construction of Computer Systems.


Digital Logic and Boolean Algebra: Digital Logic Gates and Circuits. Karnaugh maps. Use of a digital logic circuit simulator.


Components of a CPU. Processor Model. Fetch execute cycle. hardware interrupts. Assembly Language Programming : Relationship between high level Languages and Assembler. Instruction sets. Registers. Debugging. Discrete Mathematics: Matrices and Vectors.


Matrices as linear transforms. Functions: definition, properties. Sets: subsets, set algebra, Logic: Propositions, Predicates, propositional algebra, proof of simple results.


This unit would also help us Building sophisticated digital circuits using the fundamental building blocks of digital logic and boolean algebra. Writing, running and debugging a program in assembler language. Applying our knowledge in mathematical structures and techniques to solve simple problems which are based on the computing curriculum. Constructing proofs and logical arguments. Using mathematical terminology and notation to support our computing studies.


Covered so far

  • Logic gates and simple combinational circuits.

  • Combinational circuits that perform some useful functions within a computational machine.

  • The elements within an ALU and flip-flop, D-Type flip flop memory and register storage.

  • Karnaugh Maps - a graphical approach to applying Boolean Laws.

  • The general architecture of a computer and the Fetch-Execute cycle.

  • In Mathematics we covered Sets, Matrices, Vectors, and different Laws.

  • Logisim software used to draw circuit diagrams and solving other problems.

Programming


Overview of the Unit

Learning to Program in Java - An Object Oriented Language.


  • Developing a simple vocabulary of commands,syntax and programming constructs using Processing (graphical java, simple development environment).

  • Developing our problem solving skills.

  • Moving to console applications (with a more professional Java development environment) refining and building on our knowledge of programming constructs.

On successful completion of this Unit we will be able to:


  • Apply the main structuring features of the chosen high level programming language(s) to solve a variety of problems.

  • Design well-structured solutions to a problems of varying complexity using appropriate methods.

  • Implement well-structured solutions to a variety of problems using the appropriate techniques and a high-level programming language .

  • Apply object oriented design to model a variety of real world (type) problems.

  • Construct and apply suitable software tests.

Covered so far

  • Variables and expressions, drawing commands - line, point, ellipse.

  • For Loop (repeat a set number of times), while loops (repeat while something is true), Boolean expressions (tests), Pseudocode - problem solving steps in English.

  • Procedures (writing a custom command), passing parameters (information to the procedure), modular blocks of code - easy to understand code. Global variables (available throughout program), local variables (available locally within a procedure).

  • Setup and Draw events, animation, keyPressed event, if conditions.

  • Top down design, writing and using functions color type, get function command.

  • Using a Class, Objects members, methods.
